¿Cómo funciona el programa?

Una vez se lanza el programa el usuario visualizará una imagen (Euros.jpg) 
donde podrá poner el número de puntos que quiera mediante clicks de ratón.
Cada vez que el usuario llegue a hacer 4 o un múltiplo de 4 puntos, el 
polígono con los tres puntos anteriores plazados se cerrará de manera 
automática. De esta manera, una vez el usuario haya dibujado el número de 
regiones que desee al clicar la tecla 'S', se guardarán todas esas regiones
dibujadas.
